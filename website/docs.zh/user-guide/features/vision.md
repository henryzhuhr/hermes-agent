# 视觉与图片粘贴

Hermes 支持多模态视觉输入。你可以直接把图片从剪贴板贴到 CLI，让模型分析、描述或配合其他任务使用。

:::note 中文整理版
本页是 `website/docs/user-guide/features/vision.md` 的中文整理版。英文原文：[/docs/user-guide/features/vision](/docs/user-guide/features/vision)
:::

## 典型流程

1. 复制图片
2. 通过 `/paste` 或快捷键附加
3. 输入问题并发送
4. 图片以 vision block 形式发给模型

## 支持方式

- `/paste`
- 部分快捷键粘贴
- 平台相关剪贴板方式
