# 图像生成

Hermes 可以用 FAL.ai 的 FLUX 2 Pro 从文本生成图片，并自动通过 Clarity Upscaler 做 2x 放大。

:::note 中文整理版
本页是 `website/docs/user-guide/features/image-generation.md` 的中文整理版。英文原文：[/docs/user-guide/features/image-generation](/docs/user-guide/features/image-generation)
:::

## 基本流程

1. 获取 `FAL_API_KEY`
2. 写入 `~/.hermes/.env`
3. 安装客户端依赖
4. 让 Hermes 调用图像生成工具

## 适合什么

- 概念图
- 插画 / 视觉草图
- 封面或社媒素材初稿
