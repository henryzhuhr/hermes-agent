# Web Dashboard

Web Dashboard 提供一个本地浏览器界面，让你不必手工改 YAML 或频繁敲 CLI 命令，就能管理 Hermes 安装。

:::note 中文整理版
本页是 `website/docs/user-guide/features/web-dashboard.md` 的中文整理版。英文原文：[/docs/user-guide/features/web-dashboard](/docs/user-guide/features/web-dashboard)
:::

## 它能做什么

- 看状态
- 改配置
- 管理 API keys
- 浏览 sessions
- 看日志
- 看 analytics
- 管理 cron
- 管理 skills

## 启动方式

```bash
hermes dashboard
```

默认在本机开启本地服务并打开浏览器。

## 适合谁

- 不想频繁手工改配置文件
- 想可视化检查 session 和日志
- 想做本地管理面板
