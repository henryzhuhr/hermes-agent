# Honcho 记忆

Honcho 是 Hermes 的一种 memory provider 插件。它在内建记忆之上增加更强的用户建模和会话后推理能力。

:::note 中文整理版
本页是 `website/docs/user-guide/features/honcho.md` 的中文整理版。英文原文：[/docs/user-guide/features/honcho](/docs/user-guide/features/honcho)
:::

## Honcho 增加了什么

相比简单 key-value 或文件记忆，Honcho 更偏：

- 用户偏好建模
- 长期目标理解
- 交流风格建模
- 对会话后信息做推理整理

## 它属于什么

Honcho 现在是 [Memory Providers](/docs.zh/user-guide/features/memory-providers) 体系的一部分，不再是完全独立的特殊路径。
