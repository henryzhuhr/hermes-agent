# Skins 与主题

Skin 控制的是 Hermes CLI 的视觉外观，不是对话人格。它影响 banner、spinner、响应框标签、branding 文本和工具前缀等。

:::note 中文整理版
本页是 `website/docs/user-guide/features/skins.md` 的中文整理版。英文原文：[/docs/user-guide/features/skins](/docs/user-guide/features/skins)
:::

## 核心区别

- Personality：说话风格
- Skin：界面样式

## 能改什么

- 颜色
- spinner face / verb
- branding
- tool emoji
- ASCII banner

## 自定义方式

你可以在 `~/.hermes/skins/` 下放 YAML 自定义 skin。
