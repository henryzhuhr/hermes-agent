# 语音与 TTS

Hermes 支持文本转语音和语音消息转写，可用于 CLI 与多种消息平台。

:::note 中文整理版
本页是 `website/docs/user-guide/features/tts.md` 的中文整理版。英文原文：[/docs/user-guide/features/tts](/docs/user-guide/features/tts)
:::

## TTS

支持多种 provider，例如：

- Edge TTS
- ElevenLabs
- OpenAI / 兼容端点
- 其他配置好的 TTS 提供方

## STT

也支持多种语音转写 provider，并带 fallback 行为。

## 适合什么

- Telegram / Discord 语音回复
- 移动端语音交互
- 可访问性需求
