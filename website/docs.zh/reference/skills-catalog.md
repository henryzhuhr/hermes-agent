# 内置 Skills 目录

Hermes 安装时会把一大批内置 skills 复制到 `~/.hermes/skills/`。这页给出中文导航版目录，帮助你按主题找到已有能力。

:::note 中文速查版
本页是 `website/docs/reference/skills-catalog.md` 的中文速查版。原英文页包含完整分类与逐项列表：[/docs/reference/skills-catalog](/docs/reference/skills-catalog)
:::

## 主要分类

内置 skills 大致覆盖这些方向：

- Apple / macOS
- autonomous-ai-agents
- data-science
- creative
- devops
- email
- gaming
- github
- mcp
- media
- mlops
- note-taking
- productivity
- research
- red-teaming
- smart-home
- social-media
- software-development

## 怎么查看本机已安装内容

```bash
hermes skills list
/skills
```

## 怎么理解这页

- 这是“仓库里附带什么”的目录
- 具体能力和触发方式以各 skill 自身 `SKILL.md` 为准
- Optional skills 另见 [Optional Skills Catalog](/docs.zh/reference/optional-skills-catalog)
