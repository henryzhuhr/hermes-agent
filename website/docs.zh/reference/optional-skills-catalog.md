# 可选 Skills 目录

官方可选 skills 位于仓库 `optional-skills/` 中，但不会默认启用。你需要显式安装。

:::note 中文速查版
本页是 `website/docs/reference/optional-skills-catalog.md` 的中文速查版。原英文页包含完整分类与逐项清单：[/docs/reference/optional-skills-catalog](/docs/reference/optional-skills-catalog)
:::

## 安装语法

```bash
hermes skills install official/<category>/<skill>
```

例如：

```bash
hermes skills install official/blockchain/solana
```

## 主要分类

官方可选 skills 目前覆盖：

- autonomous-ai-agents
- blockchain
- communication
- creative
- devops
- email
- health
- mcp
- migration
- mlops
- productivity
- research
- security

## 什么时候看这页

- 你想扩展 Hermes，但不想自己写 skill
- 你想先看官方维护的可选技能库
