# 集成

Hermes 会和外部系统深度耦合工作：模型推理、MCP 工具服务器、编辑器、消息平台、语音和程序化接口都属于“集成层”。这一层决定 Hermes 到底能接什么、跑在哪。

:::note 中文整理版
本页是 `website/docs/integrations/index.md` 的中文整理版。英文原文：[/docs/integrations/index](/docs/integrations/index)
:::
